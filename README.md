# Dependencies

 1. Install Clojure: https://clojure.org/guides/getting_started
 2. Install Yarn: https://www.yarnpkg.com/en/docs/install
 3. Run `yarn` to install dependencies
 4. Run ./dev to start devving

## Run

``` shell
yarn install

yarn watch
```

## Clean

``` shell
yarn clean
```

## Release

``` shell
yarn release
```
